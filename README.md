# Code Mule Front End
Go to root directoy and `npm i`.

ENV variables need to be sourced while building.

## Install Element UI Theme Generator
[Link for instructions](https://element.eleme.io/#/en-US/component/custom-theme)

Install element theme generator
`npm i element-theme -g`

Install element chalk theme
`npm i element-theme-chalk -D`

Generate template files with default filenames
`et -i`

When template files are available build with 
`et`

a `theme` folder will be generated.
